The Simpsons Blog esta basado en la serie animada "The Simpsons", en este Blog podras conectarte con una comunidad de usuarios las cuales comparten contenido de imagenes con referencias a esta famosa serie.
Ademas como usuario podras realizar comentarios en las publicaciones de los usuarios.

Este sitio WEB fue creado con el framework de PHP Laravel.
